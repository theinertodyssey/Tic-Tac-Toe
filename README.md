# Tic-Tac-Toe
GDG project about tic-tac-toe.
Web Application LInk: https://tic-tac-toe-production-2b8d.up.railway.app/
The above link will land you to the web application.
<img width="1903" height="1009" alt="Screenshot from 2025-08-29 14-50-57" src="https://github.com/user-attachments/assets/a509a919-6d3b-42dd-9424-f69b7033e5ee" />

The web application has following features:-

    • Local and Multiplayer Mode.
        • Local Mode: player vs player (PVP) & CPU (with different level of difficulty)
        • Multiplayer Mode: The user can play with other player remotely via 6-digit alpha-numeric code.
    • Keep tracks of wins, loss, and draws.
        • Keeps the detailed match history and basic match stats.
    • 3 Different theme mode.
